# 웹 개발 가이드

웹 개발은 프론트엔드와 백엔드로 나뉩니다.

## 프론트엔드

- **HTML**: 웹 페이지 구조
- **CSS**: 스타일링
- **JavaScript**: 동적 기능

### 인기 프레임워크

- React
- Vue.js
- Angular

## 백엔드

서버 사이드 개발을 담당합니다:

- **Node.js**: JavaScript 런타임
- **Django**: Python 웹 프레임워크
- **Flask**: 경량 Python 프레임워크
- **Express**: Node.js 프레임워크

## 데이터베이스

- MySQL
- PostgreSQL
- MongoDB

## API 설계

RESTful API와 GraphQL이 널리 사용됩니다.

## 배포

- Docker
- AWS
- Heroku
- Vercel
