# NumPy와 배열 연산

NumPy는 Python의 강력한 수치 계산 라이브러리입니다.

## NumPy 소개

**NumPy**는 다차원 배열 객체와 다양한 수학 함수를 제공합니다.

### 주요 기능

- 빠른 배열 연산
- 브로드캐스팅
- 선형대수 연산
- 푸리에 변환

## 배열 생성

```python
import numpy as np

# 1차원 배열
arr1d = np.array([1, 2, 3, 4, 5])

# 2차원 배열
arr2d = np.array([[1, 2], [3, 4]])

# 특수 배열
zeros = np.zeros((3, 3))
ones = np.ones((2, 2))
```

## 배열 연산

NumPy는 벡터화된 연산을 지원합니다:

- 요소별 연산
- 행렬 곱셈
- 통계 함수
